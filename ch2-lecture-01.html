Lecture-1: Introduction to Function

1. What is a Function?

Intuition

A function is a rule that takes an input and produces exactly one output. Think of it like a machine: you feed an input value in, the machine applies a rule, and out comes a single output.

Formal Definition

A function \( f \) from a set \( A \) to a set \( B \) is written

\( f: A \to B \)

and for each \( x \in A \), the rule assigns a unique element \( f(x) \in B \).

  • Domain (\(A\)): the set of inputs.
  • Codomain (\(B\)): the set of possible outputs we allow.
  • Range (or image): the set of actual outputs, \( f(A) = \{ f(x): x \in A \} \subseteq B \).
Example: If \( f(x) = x^2 \) and domain is \( \mathbb{R} \), codomain is \( \mathbb{R} \), then range is \( [0,\infty) \).

2. Mapping Diagram

Let \( A = \{1,2,3\}, \; B = \{a,b,c,d\} \), define \( f \) by:

\( f(1)=a, \; f(2)=b, \; f(3)=a \).

Domain A        Codomain B
  {1}  ------>  {a}
  {2}  ------>  {b}
  {3}  ------>  {a}

Range is \( f(A) = \{a, b\} \). Codomain is \( B = \{a,b,c,d\} \).

3. Function vs Relation

A relation between \( A \) and \( B \) is any set of ordered pairs \( (x,y) \) with \( x \in A, y \in B \).

A relation is a function iff every \( x \in A \) appears in exactly one pair \( (x,y) \).

Not a function: \( R = \{(1,2), (1,3)\} \). Input 1 maps to two outputs → not a function.

4. Domain, Codomain, Range — Precise Meanings

  • Domain (\( \mathrm{Dom}(f) \)): all inputs where \( f \) is defined.
  • Codomain: declared possible outputs.
  • Range: actual outputs, always \( f(A) \subseteq B \).
Example: \( f(x) = \frac{1}{x-2} \). Domain: \( \mathbb{R} \setminus \{2\} \). Codomain: \( \mathbb{R} \). Range: \( \mathbb{R} \setminus \{0\} \).

5. Properties of Functions

Injective (One-to-One)

\( f \) is injective if different inputs give different outputs.

Example: \( f(x) = 2x \) on \( \mathbb{R} \).

Surjective (Onto)

\( f \) is surjective if every element of codomain is an output.

Example: \( f(x) = x^3 \) from \( \mathbb{R} \to \mathbb{R} \).

Bijective

If \( f \) is both injective and surjective, it is bijective and has an inverse.

Example: \( f(x) = 2x+3 \) is bijective. Inverse: \( f^{-1}(y) = \frac{y-3}{2} \).

Composition

If \( f: A \to B \), \( g: B \to C \), then composition \( g \circ f : A \to C \), with rule \( (g\circ f)(x) = g(f(x)) \).

6. Preimage

For \( S \subseteq B \), preimage is \( f^{-1}(S) = \{ x \in A : f(x) \in S \} \).

Example: \( f(x) = x^2, \; S=[1,4] \). Then \( f^{-1}(S) = [-2,-1]\cup[1,2] \).

7. Functions of One Variable

A function of one variable is a function whose domain is a subset of \( \mathbb{R} \) and output is real.

Graph: \( G = \{ (x,y): x\in D, y=f(x) \} \subseteq \mathbb{R}^2 \).

  • Polynomial: \( f(x) = x^3 - 2x + 1 \), Domain = Range = \( \mathbb{R} \).
  • Rational: \( f(x) = \tfrac{1}{x-2} \), Domain = \( \mathbb{R}\setminus\{2\} \).
  • Exponential: \( f(x)=e^x \), Range = \( (0,\infty) \).
  • Logarithm: \( f(x)=\ln x \), Domain = \( (0,\infty) \).
  • Trig: \( f(x)=\sin x \), Range = \([-1,1]\).

8. Graph Transformations

  • Vertical shift: \( y=f(x)+k \).
  • Horizontal shift: \( y=f(x-h) \).
  • Stretch/Compress: \( y=af(x), y=f(bx) \).
  • Reflection: \( y=-f(x) \) (across x-axis).
Example: \( f(x)=x^2 \). Then \( f(x-1)=(x-1)^2 \) shifts right by 1.

9. Symmetry & Periodicity

  • Even: \( f(-x)=f(x) \), symmetric about y-axis (e.g., \(x^2, \cos x\)).
  • Odd: \( f(-x)=-f(x) \), symmetric about origin (e.g., \(x^3, \sin x\)).
  • Periodic: \( f(x+T)=f(x) \), e.g. \( \sin x \) with period \(2\pi\).
