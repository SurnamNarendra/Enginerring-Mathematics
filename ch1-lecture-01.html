Introduction

A matrix is a rectangular arrangement of numbers in rows and columns. Each element is identified by its row and column position.

General Form

A = | a11  a12  a13 |
    | a21  a22  a23 |
    | a31  a32  a33 |

Types of Matrices

1. Row Matrix

[1  2  3  4]

2. Column Matrix

| 5 |
| 6 |
| 7 |

3. Square Matrix

| 1  2  3 |
| 4  5  6 |
| 7  8  9 |

4. Diagonal Matrix

| 4  0  0 |
| 0  5  0 |
| 0  0  6 |

5. Identity Matrix

| 1  0  0 |
| 0  1  0 |
| 0  0  1 |

6. Zero Matrix

| 0  0  0 |
| 0  0  0 |
| 0  0  0 |

7. Symmetric Matrix

| 1  2  3 |
| 2  5  6 |
| 3  6  9 |

8. Skew-Symmetric Matrix

| 0  -2  3 |
| 2   0 -4 |
| -3  4  0 |

9. Upper Triangular Matrix

| 1  2  3 |
| 0  4  5 |
| 0  0  6 |

10. Lower Triangular Matrix

| 7  0  0 |
| 8  9  0 |
| 1  2  3 |

11. Scalar Matrix

| 5  0  0 |
| 0  5  0 |
| 0  0  5 |

        | -2 -2 |

Multiplication

A = | 1  2 |
    | 3  4 |

B = | 2  0 |
    | 1  2 |

Applications

  • Civil Engineering: Structural analysis of buildings and bridges.
  • Electrical Engineering: Circuit analysis using mesh and nodal equations.
  • Computer Science: Graphics transformations, image processing, AI.
  • Mechanical Engineering: Vibration analysis and robotics kinematics.
  • Aerospace Engineering: Flight dynamics and control systems.
