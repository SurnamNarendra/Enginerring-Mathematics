Lecture-1: Introduction to Functions

Lecture-2: Limits and Contiunity

In this lecture we deepen the idea of limits (including left and right limits), learn the formal epsilon–delta definition, work several step-by-step examples, and classify types of continuity and discontinuity.

1. Intuitive idea of a limit

When we write \(\displaystyle \lim_{x\to a} f(x)=L\) we mean: as \(x\) gets arbitrarily close to the number \(a\) (but not necessarily equal to \(a\)), the outputs \(f(x)\) get arbitrarily close to \(L\).

Important: The value \(f(a)\) (if defined) is not part of the definition of the limit — the limit cares only about values near \(a\).

2. One-sided limits (Left-hand and Right-hand)

For a point \(a\):

  • Left-hand limit: \(\displaystyle \lim_{x\to a^-} f(x)\) means \(x\) approaches \(a\) from values < \(a\).
  • Right-hand limit: \(\displaystyle \lim_{x\to a^+} f(x)\) means \(x\) approaches \(a\) from values > \(a\).

The two-sided limit \(\lim_{x\to a} f(x)\) exists iff both one-sided limits exist and are equal:

\(\lim_{x\to a^-} f(x) = \lim_{x\to a^+} f(x) = L \Longrightarrow \lim_{x\to a} f(x)=L.\)

Example — step function
Let \[ f(x)=\begin{cases} 1 & x<0,\\ 2 & x\ge 0. \end{cases} \] Then \(\lim_{x\to 0^-} f(x)=1\) but \(\lim_{x\to 0^+} f(x)=2\). Since they differ, \(\lim_{x\to 0} f(x)\) does not exist.

3. Formal (ε–δ) definition of limit — explained

We say \(\displaystyle \lim_{x\to a} f(x)=L\) if:

For every \(\varepsilon>0\) there exists \(\delta>0\) such that whenever \(0<|x-a|<\delta\) we have \(|f(x)-L|<\varepsilon\).

Informal unpacking: choose how close you want \(f(x)\) to be to \(L\) (that's \(\varepsilon\)). The definition guarantees an interval \((a-\delta,a+\delta)\) around \(a\) (except possibly \(a\) itself) where all function values are within that desired closeness.

Remark: We require \(0<|x-a|<\delta\) (exclude \(x=a\)) because the value at \(a\) doesn't matter for the limit.
Illustrative ε–δ example (linear function)
Show \(\displaystyle \lim_{x\to 2} (3x+1)=7\) via ε–δ idea.
For given \(\varepsilon>0\) we want \(|(3x+1)-7|<\varepsilon\). Compute: \[ |3x+1-7|=|3x-6|=3|x-2|. \] If we ensure \(3|x-2|<\varepsilon\) then the requirement holds. So choose \(\delta=\varepsilon/3\). Then whenever \(0<|x-2|<\delta\) we have \(|3x+1-7|<\varepsilon\). Thus limit holds.

4. Basic limit techniques (with worked examples)

(A) Direct substitution
If \(f\) is a polynomial or rational expression with no zero denominator at \(a\), you can substitute \(x=a\).
Example: \(\lim_{x\to 3} (2x^2-1)=2(3)^2-1=17.\)
(B) Factor & cancel (removable form)
Example: \(\displaystyle \lim_{x\to 2} \frac{x^2-4}{x-2}\).
Factor numerator: \(x^2-4=(x-2)(x+2)\). For \(x\neq2\) the fraction equals \(x+2\). Thus \[ \lim_{x\to 2} \frac{x^2-4}{x-2} = \lim_{x\to 2} (x+2)=4. \] This was a removable discontinuity if considered as a function with domain excluding \(2\).
(C) Rationalize (root expressions)
Example: \(\displaystyle \lim_{x\to 0} \frac{\sqrt{1+x}-1}{x}\).
Multiply numerator and denominator by the conjugate \(\sqrt{1+x}+1\): \[ \frac{\sqrt{1+x}-1}{x}\cdot\frac{\sqrt{1+x}+1}{\sqrt{1+x}+1} =\frac{(1+x)-1}{x(\sqrt{1+x}+1)}=\frac{x}{x(\sqrt{1+x}+1)}=\frac{1}{\sqrt{1+x}+1}. \] Now substitute \(x\to0\): limit = \(1/(1+1)=1/2\).
(D) Trigonometric limit
A very useful limit: \[ \lim_{x\to 0}\frac{\sin x}{x}=1. \] (Often proved with geometry or squeeze theorem. Accept as a standard limit.)
(E) Limits at infinity — leading terms
Example: \(\displaystyle \lim_{x\to\infty} \frac{3x^2+2x-1}{5x^2-7}=\frac{3}{5}\) because highest-degree terms dominate.

5. One-sided limits — practical computations

Example 1 (one-sided rational limit)
Compute \(\lim_{x\to 1^-}\frac{x^2-1}{x-1}\) and \(\lim_{x\to 1^+}\frac{x^2-1}{x-1}\).
For \(x\ne1\), \(\frac{x^2-1}{x-1}=x+1\). So both one-sided limits equal \(2\). Therefore two-sided limit = 2.
Example 2 (piecewise function)
Let \[ f(x)=\begin{cases} 2x+1 & x<0\\ x^2 & x\ge 0. \end{cases} \] Find one-sided limits at \(0\).
Left-hand: \(\lim_{x\to 0^-} f(x)=\lim_{x\to0^-} (2x+1) = 1\). Right-hand: \(\lim_{x\to 0^+} f(x)=\lim_{x\to0^+} x^2 = 0\). They differ → two-sided limit does not exist at \(0\).

6. Continuity — definition & intuitive checks

Function \(f\) is continuous at \(a\) if

  1. \(f(a)\) is defined;
  2. \(\lim_{x\to a} f(x)\) exists;
  3. \(\lim_{x\to a} f(x)=f(a)\).

If \(f\) is continuous at every point of its domain we say \(f\) is continuous on that domain (or continuous everywhere if domain is \(\mathbb{R}\)).

Common continuous functions: polynomials, rational functions (on domain where denominator ≠ 0), exponential, logarithms (on their domain), trig functions (on their domain). These are building blocks — combinations using algebraic rules remain continuous where defined.
Example — removable discontinuity
\(f(x)=\dfrac{x^2-1}{x-1}\) (as a function with domain \(x\ne1\)) has \(\lim_{x\to1}f(x)=2\) but \(f(1)\) is not defined. If we define \(g(x)=f(x)\) for \(x\ne1\) and \(g(1)=2\), then \(g\) is continuous at \(1\).
Example — infinite discontinuity
\(f(x)=\dfrac{1}{x}\) has no finite limit as \(x\to0\); instead \(\lim_{x\to0^+} f(x)=+\infty\) and \(\lim_{x\to0^-} f(x)=-\infty\).

7. Classification of discontinuities (summary)

  • Removable: limit exists but function undefined or defined to a different value (a hole that can be 'filled').
  • Jump: left and right limits exist but are unequal (a finite jump in value).
  • Infinite: one or both one-sided limits are infinite (vertical asymptote).
  • Essential / oscillatory: values oscillate and no limit exists (e.g., \(f(x)=\sin(1/x)\) at \(0\)).

8. Worked longer examples (step-by-step)

Example A — limit using conjugate
Compute \(\displaystyle \lim_{x\to 0} \frac{\sqrt{1+2x}-1}{x}\).
Multiply top & bottom by conjugate: \[ \frac{\sqrt{1+2x}-1}{x}\cdot\frac{\sqrt{1+2x}+1}{\sqrt{1+2x}+1} =\frac{(1+2x)-1}{x(\sqrt{1+2x}+1)}=\frac{2x}{x(\sqrt{1+2x}+1)} =\frac{2}{\sqrt{1+2x}+1}. \] Substitute \(x\to 0\): result \(=2/(1+1)=1\).
Example B — one-sided infinite limit
Compute \(\lim_{x\to 2^-}\frac{1}{x-2}\) and \(\lim_{x\to 2^+}\frac{1}{x-2}\).
For \(x\to2^-\) denominator \(x-2\) is small negative → fraction → \(-\infty\). For \(x\to2^+\) denominator small positive → fraction → \(+\infty\). So vertical asymptote at \(x=2\).
Example C — oscillatory (no limit)
Consider \(f(x)=\sin\!\left(\tfrac{1}{x}\right)\) for \(x\ne0\). As \(x\to 0\) the argument \(1/x\) runs through arbitrarily large values → the sine oscillates between \(-1\) and \(1\) infinitely often. Thus \(\lim_{x\to 0}\sin(1/x)\) does not exist.

9. Practice problems (with answers)

  1. Problem: Compute \(\displaystyle \lim_{x\to 1}\frac{x^3-1}{x-1}\).
    Factor \(x^3-1=(x-1)(x^2+x+1)\). For \(x\neq1\) the fraction = \(x^2+x+1\). Substitute → \(1+1+1=3\).
  2. Problem: Evaluate one-sided limits of \(f(x)=\dfrac{|x|}{x}\) at \(0\).
    For \(x>0\), \(|x|/x=1\). For \(x<0\), result = -1. So \(\lim_{x\to 0^+} =1,\ \lim_{x\to 0^-}=-1\). Two-sided limit does not exist.
  3. Problem: Is \(g(x)=\begin{cases}\dfrac{x^2-4}{x-2}&x\ne2\\5&x=2\end{cases}\) continuous at \(2\)?
    For \(x\ne2\), expression simplifies to \(x+2\). Limit as \(x\to2\) is 4. But \(g(2)=5\ne4\). So not continuous (removable discontinuity that wasn't filled correctly).
  4. Problem: Compute \(\displaystyle \lim_{x\to 0} \frac{\sin 3x}{x}\).
    Write \(\frac{\sin 3x}{x} = 3\cdot \frac{\sin 3x}{3x}\). As \(x\to0\), inner limit = 1, so overall limit = 3.
  5. Problem: Determine whether \(f(x)=\tan x\) is continuous at \(x=\pi/2\).
    At \(x=\pi/2\) tangent has vertical asymptote → one-sided limits are infinite → not continuous.

10. Quick summary & tips

  • To check a two-sided limit, check both one-sided limits.
  • If you get \(0/0\) try algebra (factor, rationalize) — it often indicates a removable form.
  • \(\sin x / x \to 1\) as \(x\to0\) is a very useful trig limit to memorize.
  • Continuity: check the three conditions (existence of value, existence of limit, equality).
  • Graphing helps: draw or imagine the graph near the point to see holes, jumps, or asymptotes.
