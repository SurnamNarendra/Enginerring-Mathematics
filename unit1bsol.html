Practice Set-2 — Solutions

Section A — Eigenvalues & Eigenvectors

2×2 Matrices

  1. Problem: \(A=\begin{bmatrix}2&1\\1&2\end{bmatrix}\). Find eigenvalues and eigenvectors.
    Characteristic polynomial: \((\lambda-3)(\lambda-1)=0\). Eigenvalues: \(\lambda_1=3,\ \lambda_2=1\). Eigenvectors: \[ \lambda=3:\; v_1=\begin{bmatrix}1\\1\end{bmatrix},\qquad \lambda=1:\; v_2=\begin{bmatrix}-1\\1\end{bmatrix}. \]
  2. Problem: \(B=\begin{bmatrix}4&-2\\1&1\end{bmatrix}\).
    Char. poly: \((\lambda-3)(\lambda-2)\). Eigenvalues: \(\lambda=3,2\). Eigenvectors (one choice each): \[ \lambda=2:\; v=\begin{bmatrix}1\\1\end{bmatrix},\qquad \lambda=3:\; v=\begin{bmatrix}2\\1\end{bmatrix}. \]
  3. Problem: \(C=\begin{bmatrix}0&1\\-4&-5\end{bmatrix}\).
    Char. poly: \((\lambda+4)(\lambda+1)\). Eigenvalues: \(\lambda=-4,\,-1\). Eigenvectors: \[ \lambda=-4:\; v=\begin{bmatrix}-\tfrac14\\1\end{bmatrix},\qquad \lambda=-1:\; v=\begin{bmatrix}-1\\1\end{bmatrix}. \] (Any nonzero scalar multiple is acceptable.)
  4. Problem: \(D=\begin{bmatrix}3&2\\0&3\end{bmatrix}\).
    Characteristic polynomial: \((\lambda-3)^2\). Eigenvalue \(\lambda=3\) with algebraic multiplicity 2. Solve \((D-3I)v=0\) gives eigenvectors proportional to \(\begin{bmatrix}1\\0\end{bmatrix}\) only — a single independent eigenvector. Conclusion: Not enough independent eigenvectors (defective); \(D\) is not diagonalizable.
  5. Problem: \(E=\begin{bmatrix}1&2\\2&1\end{bmatrix}\).
    Char. poly: \((\lambda-3)(\lambda+1)\). Eigenvalues: \(\lambda=3,\,-1\). Eigenvectors: \[ \lambda=3:\; v=\begin{bmatrix}1\\1\end{bmatrix},\qquad \lambda=-1:\; v=\begin{bmatrix}1\\-1\end{bmatrix}. \]

3×3 Matrices

  1. Problem: \(F=\begin{bmatrix}1&0&0\\0&2&1\\0&0&2\end{bmatrix}\).
    This is block/upper-triangular → eigenvalues are diagonal entries: \(1,2,2\). Eigenvectors: \[ \lambda=1:\; v=\begin{bmatrix}1\\0\\0\end{bmatrix},\qquad \lambda=2:\; \text{solve }(F-2I)v=0\Rightarrow v=\begin{bmatrix}0\\1\\0\end{bmatrix} \text{ and a generalized eigenvector exists due to the Jordan block}. \] There is only one independent eigenvector for \(\lambda=2\) coming from the 2×2 Jordan block; \(F\) is not diagonalizable (it has a size 2 Jordan block).
  2. Problem: \(G=\begin{bmatrix}2&0&0\\0&3&4\\0&0&3\end{bmatrix}\).
    Upper-triangular ⇒ eigenvalues: \(2,3,3\). Eigenvectors: \[ \lambda=2:\ v=\begin{bmatrix}1\\0\\0\end{bmatrix},\qquad \lambda=3:\ \text{one independent eigenvector }v=\begin{bmatrix}0\\1\\0\end{bmatrix}. \] (Because the \(2\times2\) lower-right block is a Jordan block, there is only one eigenvector for \(\lambda=3\); not diagonalizable.)
  3. Problem: \(H=\begin{bmatrix}4&0&0\\0&2&1\\0&1&2\end{bmatrix}\).
    Block diagonal in effect: eigenvalue \(4\) from first row. For the \(2\times2\) block \(\begin{bmatrix}2&1\\1&2\end{bmatrix}\) eigenvalues \(3\) and \(1\). So eigenvalues: \(4,3,1\). Corresponding eigenvectors: \[ \lambda=4:\; v=\begin{bmatrix}1\\0\\0\end{bmatrix},\quad \lambda=3:\; v=\begin{bmatrix}0\\1\\1\end{bmatrix},\quad \lambda=1:\; v=\begin{bmatrix}0\\-1\\1\end{bmatrix}. \]
  4. Problem: \(I=\begin{bmatrix}0&1&0\\0&0&1\\1&0&0\end{bmatrix}\).
    Characteristic polynomial: \((\lambda-1)(\lambda^2+\lambda+1)\). Eigenvalues: \(\lambda=1\) and the two complex cube-root pair \(\lambda = e^{\pm i\pi/3} = -\tfrac12 \pm i\tfrac{\sqrt3}{2}\). Real eigenvector for \(\lambda=1\): \[ v=\begin{bmatrix}1\\1\\1\end{bmatrix}. \] The other two eigenvectors are complex (corresponding to the complex eigenvalues).
  5. Problem: \(J=\begin{bmatrix}2&-1&0\\-1&2&-1\\0&-1&2\end{bmatrix}\) (tridiagonal).
    Characteristic polynomial factors to produce eigenvalues: \[ \lambda_1=2,\quad \lambda_{2}=2-\sqrt2,\quad \lambda_3=2+\sqrt2. \] Corresponding eigenvectors (one choice each): \[ \lambda=2:\ v=\begin{bmatrix}-1\\0\\1\end{bmatrix},\quad \lambda=2-\sqrt2:\ v=\begin{bmatrix}1\\\sqrt2\\1\end{bmatrix},\quad \lambda=2+\sqrt2:\ v=\begin{bmatrix}1\\-\sqrt2\\1\end{bmatrix}. \]

Section B — Cayley–Hamilton Theorem

  1. Problem: Verify Cayley–Hamilton for \(A=\begin{bmatrix}2&1\\1&2\end{bmatrix}\).
    Char. polynomial \(p(\lambda)=\lambda^2-4\lambda+3\). CH says \(p(A)=A^2-4A+3I=0\). Compute quickly: \[ A^2=\begin{bmatrix}2&1\\1&2\end{bmatrix}^2=\begin{bmatrix}5&4\\4&5\end{bmatrix}, \] so \[ A^2-4A+3I=\begin{bmatrix}5&4\\4&5\end{bmatrix}-4\begin{bmatrix}2&1\\1&2\end{bmatrix}+3\begin{bmatrix}1&0\\0&1\end{bmatrix} =\begin{bmatrix}0&0\\0&0\end{bmatrix}. \] Verified.
  2. Problem: Use CH to compute \(A^3\) for \(A=\begin{bmatrix}1&1\\0&1\end{bmatrix}\).
    The matrix is upper-triangular with eigenvalue \(1\) (double). Direct power formula works: powers add the superdiagonal: \[ A^2=\begin{bmatrix}1&2\\0&1\end{bmatrix},\qquad A^3=\begin{bmatrix}1&3\\0&1\end{bmatrix}. \] (Cayley–Hamilton also gives a relation but direct computation is simplest here.)
  3. Problem: Use CH to find \(A^{-1}\) for \(A=\begin{bmatrix}3&2\\1&1\end{bmatrix}\).
    Char. poly: \(\lambda^2-4\lambda+1=0\). CH: \(A^2-4A+I=0\). Multiply on right by \(A^{-1}\): \[ A-4I + A^{-1}=0\quad\Rightarrow\quad A^{-1}=4I-A. \] So \[ A^{-1}=4\begin{bmatrix}1&0\\0&1\end{bmatrix}-\begin{bmatrix}3&2\\1&1\end{bmatrix} =\begin{bmatrix}1&-2\\-1&3\end{bmatrix}. \] (Matches the usual adjoint/determinant method since \(\det A=1\).)
  4. Problem: Verify CH for the \(3\times3\) companion-like matrix \(A=\begin{bmatrix}0&1&0\\0&0&1\\-6&11&-6\end{bmatrix}.\)
    Characteristic polynomial is \(p(\lambda)=\lambda^3-6\lambda^2+11\lambda-6=(\lambda-1)(\lambda-2)(\lambda-3)\). CH predicts \[ A^3-6A^2+11A-6I=0. \] If you compute these powers and form the linear combination you obtain the zero matrix — so CH holds (direct substitution gives the zero matrix).
  5. Problem: Use CH to compute \(A^4\) for \(A=\begin{bmatrix}4&3\\3&2\end{bmatrix}.\)
    One way is to compute the power directly (or reduce higher powers using CH). Direct computation yields \[ A^2=\begin{bmatrix}25&18\\18&13\end{bmatrix},\quad A^3=\begin{bmatrix}169&120\\120&85\end{bmatrix},\quad A^4=\begin{bmatrix}949&684\\684&493\end{bmatrix}. \] (You can also reduce \(A^4\) using the characteristic polynomial of degree 2 to express \(A^4\) as a linear combination of \(A\) and \(I\).)

Section C — Diagonalization

  1. Problem: Diagonalize \(A=\begin{bmatrix}5&4\\1&2\end{bmatrix}\).
    Char. poly: \(\lambda^2-7\lambda+6=(\lambda-6)(\lambda-1)\). Eigenvalues \(\lambda=6,1\). Eigenvectors: \[ \lambda=6:\ v_1=\begin{bmatrix}2\\1\end{bmatrix},\qquad \lambda=1:\ v_2=\begin{bmatrix}-1\\1\end{bmatrix}. \] Form \(P=[v_1\ v_2]=\begin{bmatrix}2&-1\\1&1\end{bmatrix}\). Then \[ P^{-1}AP=\operatorname{diag}(6,1). \]
  2. Problem: Find \(P\) so \(P^{-1}BP\) is diagonal for \(B=\begin{bmatrix}2&0\\0&3\end{bmatrix}\).
    \(B\) is already diagonal with eigenvalues \(2,3\). Choose \(P=I\) or any matrix whose columns are eigenvectors (e.g. \(P=I\)). Then \(P^{-1}BP=B\) (diagonal).
  3. Problem: Is \(C=\begin{bmatrix}1&1\\0&1\end{bmatrix}\) diagonalizable?
    Char. poly is \((\lambda-1)^2\). Only one eigenvector (span of \([1,0]^T\)). So not diagonalizable (it is a Jordan block).
  4. Problem: Diagonalize \(D=\begin{bmatrix}6&-2\\2&3\end{bmatrix}\).
    Char. poly: \(\lambda^2-9\lambda+22\). Solve to get eigenvalues \[ \lambda=\frac{9\pm\sqrt{81-88}}{2}=\frac{9\pm i\sqrt7}{2} \] (complex). Because eigenvalues are distinct (though complex), the matrix is diagonalizable over \(\mathbb{C}\). Over \(\mathbb{R}\) it is not diagonalizable to a real diagonal matrix (it has complex eigenvalues).
  5. Problem: Find diagonal form of \(E=\begin{bmatrix}2&1&0\\0&2&0\\0&0&3\end{bmatrix}.\)
    Eigenvalues: \(2\) (algebraic multiplicity 2) and \(3\). For the \(2\times2\) upper-left block \(\begin{bmatrix}2&1\\0&2\end{bmatrix}\) there is only one eigenvector for \(\lambda=2\), so the full matrix is not diagonalizable over \(\mathbb{R}\). But if it were diagonalizable, the diagonal would be \(\operatorname{diag}(2,2,3)\).
