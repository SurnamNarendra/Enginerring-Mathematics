Lecture 8: Functions of Several Variables – Limits & Continuity

1. Introduction

A function of several variables maps tuples of numbers to a real number: \[ f: D \subset \mathbb{R}^n \to \mathbb{R}, \quad (x_1, x_2, ..., x_n) \mapsto f(x_1, x_2, ..., x_n) \] Most common case: two variables \(f(x,y)\). Functions of several variables are essential in physics, engineering, and optimization problems.

2. Domain, Codomain, and Range

For \(f(x,y)\):

  • Domain: set of all \((x,y)\) for which \(f(x,y)\) is defined
  • Codomain: usually \(\mathbb{R}\)
  • Range: set of actual output values of \(f\)

3. Limits of Functions of Two Variables

The limit of \(f(x,y)\) as \((x,y) \to (a,b)\) is \(L\) if: \[ \lim_{(x,y)\to(a,b)} f(x,y) = L \] means that for every path approaching \((a,b)\), \(f(x,y) \to L\).

Example 1:

Check if \(\lim_{(x,y)\to(0,0)} \frac{xy}{x^2+y^2}\) exists.

  • Along \(y=0\): \(\frac{x*0}{x^2+0} = 0\)
  • Along \(y=x\): \(\frac{x*x}{x^2+x^2} = \frac{x^2}{2x^2} = \frac{1}{2}\)
  • Different paths → limit does not exist.

4. Continuity

A function \(f(x,y)\) is continuous at \((a,b)\) if: \[ \lim_{(x,y)\to(a,b)} f(x,y) = f(a,b) \] This requires that the limit exists and equals the function value.

Example 2:

Check continuity of \(f(x,y) = x^2 + y^2\) at (1,1).

  • \(\lim_{(x,y)\to(1,1)} (x^2 + y^2) = 1^2 + 1^2 = 2 = f(1,1)\)
  • Hence, \(f\) is continuous at (1,1)

5. Left-side and Right-side Analogy

In multivariable functions, “left” and “right” are replaced by “paths” of approach. Continuity requires the same limit along all possible paths approaching the point.

Example 3:

\(f(x,y) = \frac{x^2y}{x^2+y^2}\) at (0,0).

  • Along y=0 → 0
  • Along x=0 → 0
  • Along y=x → \(\frac{x^3}{2x^2} = x/2 \to 0\)
  • All paths → limit 0 = f(0,0), so continuous at (0,0)

7. Summary

  • Function of several variables: \(f(x_1, x_2, ..., x_n)\)
  • Limit exists only if approaching along all paths gives the same value
  • Continuity: limit at a point equals function value
  • Path analysis is crucial for detecting non-existence of limits

  3. Find limits along paths y=0, x=0, y=x for \(f(x,y)=\frac{xy}{x^2+y^2}\)
