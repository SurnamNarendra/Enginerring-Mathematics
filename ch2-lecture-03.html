Lecture-3: Increasing and Decreasing Functions

Lecture 3: Increasing and Decreasing Functions

1. Introduction

One of the most important uses of derivatives is to study how a function behaves: whether it is increasing (rising) or decreasing (falling). This helps us understand growth, decay, and optimization problems in real life.

2. Definitions

  • Increasing Function: A function \(f(x)\) is said to be increasing on an interval if for any \(x_1 < x_2\), we have \(f(x_1) \leq f(x_2)\).
  • Strictly Increasing: If \(f(x_1) < f(x_2)\) for all \(x_1 < x_2\).
  • Decreasing Function: A function \(f(x)\) is decreasing on an interval if for any \(x_1 < x_2\), we have \(f(x_1) \geq f(x_2)\).
  • Strictly Decreasing: If \(f(x_1) > f(x_2)\) for all \(x_1 < x_2\).

3. Role of Derivatives

For a differentiable function \(f(x)\): - If \(f'(x) > 0\) on an interval → \(f(x)\) is increasing. - If \(f'(x) < 0\) on an interval → \(f(x)\) is decreasing. - If \(f'(x) = 0\) at some point → it may be a local maxima, minima, or a point of inflection.

4. Worked Examples

Example 1: Polynomial Function

Determine the intervals where \(f(x) = x^3 - 3x^2 + 2\) is increasing or decreasing.

Solution:

  • Differentiate: \(f'(x) = 3x^2 - 6x = 3x(x-2)\).
  • Critical points: \(x = 0, 2\).
  • Test intervals:
    • For \(x < 0\): \(f'(x) > 0\) → Increasing.
    • For \(0 < x < 2\): \(f'(x) < 0\) → Decreasing.
    • For \(x > 2\): \(f'(x) > 0\) → Increasing.

So, \(f(x)\) increases on \((-\infty,0)\) and \((2,\infty)\), decreases on \((0,2)\).

Example 2: Trigonometric Function

Find intervals of increase and decrease of \(f(x) = \sin x\) in \([0,2\pi]\).

  • \(f'(x) = \cos x\).
  • In \([0,2\pi]\):
    • \(f'(x) > 0\) on \((0,\pi/2)\) and \((3\pi/2,2\pi)\) → Increasing.
    • \(f'(x) < 0\) on \((\pi/2,3\pi/2)\) → Decreasing.

Example 3: Exponential Function

For \(f(x) = e^x\), derivative is \(f'(x) = e^x > 0\) for all real \(x\). Hence, \(f(x)\) is strictly increasing everywhere.

5. Real-Life Applications

Economics: Profit Maximization

If profit function \(P(x)\) increases up to a certain production level and then decreases, the derivative test can identify the production quantity that maximizes profit.

Physics: Motion

If displacement \(s(t)\) is given, then \(s'(t)\) = velocity. - If velocity > 0 → object moves forward (increasing position). - If velocity < 0 → object moves backward (decreasing position).

Population Growth

If population \(P(t)\) has \(P'(t) > 0\), population is growing. If \(P'(t) < 0\), population is declining.

6. Optimization Problems

Example 4: Maximize Area of Rectangle

A rectangle is inscribed under the curve \(y = 12 - x^2\) above the x-axis. Find dimensions of rectangle with max area.

  • Area \(A = 2x \cdot y = 2x(12 - x^2)\).
  • \(A(x) = 24x - 2x^3\).
  • Differentiate: \(A'(x) = 24 - 6x^2\).
  • Set = 0 → \(x^2 = 4 \Rightarrow x = 2\).
  • Max area at \(x=2, y=12-4=8\).
  • Rectangle dimensions: width \(=4\), height \(=8\), area \(=32\).

Example 5: Minimize Cost

A farmer wants to build a rectangular field with fencing on 3 sides (one side is a river). If total fencing available = 200 m, maximize area.

  • Let width = x, length = y. Fence: \(2x + y = 200\).
  • Area \(A = xy\). Substitute \(y = 200 - 2x\).
  • \(A(x) = x(200 - 2x) = 200x - 2x^2\).
  • \(A'(x) = 200 - 4x = 0 \Rightarrow x = 50\).
  • \(y = 200 - 100 = 100\).
  • Max area = \(50 \times 100 = 5000\) m².

7. Summary

  • Derivative sign test determines increasing/decreasing behavior.
  • Critical points help identify turning points.
  • Applications in economics, physics, and optimization.
