Lecture 6: Maxima and Minima

1. Introduction

Maxima and minima (extrema) of a function are the points where it reaches its highest or lowest values, either locally or globally. These are essential in optimization problems in physics, economics, and engineering.

2. Critical Points

For a differentiable function \(f(x)\), a critical point occurs where: \[ f'(x) = 0 \quad \text{or} \quad f'(x) \text{ does not exist} \] Critical points are candidates for local maxima or minima.

3. First Derivative Test

If \(f'(x)\) changes sign around a critical point \(x=c\):

  • \(f'(x)\) changes from positive to negative → local maximum
  • \(f'(x)\) changes from negative to positive → local minimum

4. Second Derivative Test

If \(f''(c)\) exists:

  • \(f''(c) > 0\) → local minimum at \(x=c\)
  • \(f''(c) < 0\) → local maximum at \(x=c\)
  • \(f''(c) = 0\) → test inconclusive

5. Worked Example 1: Polynomial

Function: \(f(x)=x^3-3x^2+2\)

  • Derivative: \(f'(x)=3x^2-6x=3x(x-2)\)
  • Critical points: \(x=0,2\)
  • Second derivative: \(f''(x)=6x-6\)
  • At \(x=0\), \(f''(0)=-6 <0\) → local maximum
  • At \(x=2\), \(f''(2)=6>0\) → local minimum
  • Local maximum: \(f(0)=2\), Local minimum: \(f(2)=-2\)

6. Worked Example 2: Optimization Problem

Find the dimensions of a rectangle with perimeter 20 m that maximize area.

  • Let length \(x\), width \(y\). Perimeter: \(2(x+y)=20 \Rightarrow y=10-x\)
  • Area: \(A(x)=x(10-x)=10x-x^2\)
  • Derivative: \(A'(x)=10-2x=0 \Rightarrow x=5\)
  • Second derivative: \(A''(x)=-2<0\) → maximum
  • Width: \(y=10-5=5\). Maximum area: \(A=25\, m^2\)

7. Worked Example 3: Real-Life Application

A company’s profit function: \(P(x)=-5x^2+150x-1000\), where \(x\) is number of units sold. Find the units for maximum profit.

  • Derivative: \(P'(x)=-10x+150=0 \Rightarrow x=15\)
  • Second derivative: \(P''(x)=-10<0\) → maximum
  • Maximum profit: \(P(15)=-5*225+2250-1000=1250\)

9. Summary

  • Critical points: solve \(f'(x)=0\) or where derivative undefined.
  • Use First Derivative Test or Second Derivative Test to classify extrema.
  • Extremum problems appear in geometry, economics, physics, and engineering.
