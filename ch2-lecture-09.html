Lecture 9: Partial Derivatives and Maxima/Minima of Multivariable Functions

1. Introduction to Partial Derivatives

A partial derivative measures how a function changes with respect to one variable while keeping other variables constant. For a function \(f(x,y)\): \[ \frac{\partial f}{\partial x} = \lim_{h\to 0} \frac{f(x+h, y) - f(x, y)}{h}, \quad \frac{\partial f}{\partial y} = \lim_{h\to 0} \frac{f(x, y+h) - f(x, y)}{h} \]

2. Higher-Order Partial Derivatives

Second-order derivatives include: \[ f_{xx} = \frac{\partial^2 f}{\partial x^2}, \quad f_{yy} = \frac{\partial^2 f}{\partial y^2}, \quad f_{xy} = \frac{\partial^2 f}{\partial x \partial y}, \quad f_{yx} = \frac{\partial^2 f}{\partial y \partial x} \] If \(f\) is smooth, then \(f_{xy} = f_{yx}\) (Clairaut's theorem).

Example 1: Find partial derivatives of \(f(x,y) = x^2y + 3xy^2\)

  • \(\frac{\partial f}{\partial x} = 2xy + 3y^2\)
  • \(\frac{\partial f}{\partial y} = x^2 + 6xy\)

3. Maxima and Minima of Functions of Two Variables

To find local maxima, minima, or saddle points:

  1. Compute first-order partial derivatives \(f_x, f_y\) and set them to zero to find critical points.
  2. Compute second-order partial derivatives \(f_{xx}, f_{yy}, f_{xy}\).
  3. Use the second derivative test: \[ D = f_{xx} f_{yy} - (f_{xy})^2 \]
    • If \(D>0\) and \(f_{xx}>0\), local minimum.
    • If \(D>0\) and \(f_{xx}<0\), local maximum.
    • If \(D<0\), saddle point.
    • If \(D=0\), test is inconclusive.

Example 2: Find and classify critical points of \(f(x,y) = x^2 + xy + y^2\)

  • \(f_x = 2x + y = 0 \Rightarrow y = -2x\)
  • \(f_y = x + 2y = 0 \Rightarrow x = -2y\)
  • Solving gives critical point: \(x=0, y=0\)
  • Second derivatives: \(f_{xx}=2, f_{yy}=2, f_{xy}=1\)
  • \(D = 2*2 - 1^2 = 3 > 0\), \(f_{xx}=2>0\) â†’ local minimum at (0,0)

4. Method of Lagrange Multipliers (Constrained Optimization)

To maximize or minimize \(f(x,y)\) subject to \(g(x,y) = 0\): \[ \nabla f = \lambda \nabla g \] where \(\nabla f = (f_x, f_y)\) and \(\nabla g = (g_x, g_y)\). Solve system of equations including the constraint.

Example 3: Maximize \(f(x,y) = xy\) subject to \(x+y=10\)

  • Constraint: \(g(x,y)=x+y-10=0\)
  • \(\nabla f = (y, x), \nabla g = (1,1)\)
  • Set \(\nabla f = \lambda \nabla g \Rightarrow y = \lambda, x = \lambda \Rightarrow x=y\)
  • Use constraint: \(x+y=10 \Rightarrow 2x=10 \Rightarrow x=y=5\)
  • Maximum value: \(f(5,5)=25\)

5. Applications

  • Economics: maximize profit or minimize cost under budget constraints
  • Engineering: stress/strain optimization, material design
  • Physics: potential energy minima, equilibrium points
  • Machine learning: optimize loss functions with multiple variables
