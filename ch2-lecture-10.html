Lecture 10: Taylor's Theorem for Multivariable Functions

1. Introduction

Taylor's theorem generalizes the linear approximation of functions to higher-order terms. For functions of several variables, it expresses \(f(x,y)\) near a point \((a,b)\) as a sum of polynomial terms plus a remainder.

2. Taylor's Theorem (Statement)

If \(f(x,y)\) has continuous partial derivatives up to order \(n+1\) near \((a,b)\), the Taylor series expansion of \(f\) around \((a,b)\) is: \[ f(x,y) = f(a,b) + f_x(a,b)(x-a) + f_y(a,b)(y-b) + \frac{1}{2!} \Big[ f_{xx}(a,b)(x-a)^2 + 2f_{xy}(a,b)(x-a)(y-b) + f_{yy}(a,b)(y-b)^2 \Big] + \cdots + R_n \] where \(R_n\) is the remainder term.

3. First and Second Order Approximations

- First-order (linear approximation): \[ f(x,y) \approx f(a,b) + f_x(a,b)(x-a) + f_y(a,b)(y-b) \] - Second-order (quadratic approximation): \[ f(x,y) \approx f(a,b) + f_x(a,b)(x-a) + f_y(a,b)(y-b) + \frac{1}{2}\Big(f_{xx}(a,b)(x-a)^2 + 2 f_{xy}(a,b)(x-a)(y-b) + f_{yy}(a,b)(y-b)^2 \Big) \]

Example 1: Second-Order Approximation

Approximate \(f(x,y) = e^{x+y}\) near (0,0)

  • First derivatives: \(f_x = e^{x+y}, f_y = e^{x+y}\), at (0,0) → 1
  • Second derivatives: \(f_{xx} = f_{xy} = f_{yy} = e^{x+y}\), at (0,0) → 1
  • Second-order Taylor expansion:
  • \[ f(x,y) \approx 1 + x + y + \frac{1}{2} (x^2 + 2xy + y^2) = 1 + x + y + \frac{1}{2}(x+y)^2 \]

4. Applications of Taylor's Theorem

  • Optimization: Quadratic approximations help classify critical points using the second derivative test.
  • Numerical methods: Approximate function values and gradients for computational solutions.
  • Physics: Small perturbation analysis, e.g., potential energy near equilibrium.
  • Engineering: Linearization of nonlinear systems for control and stability analysis.
  • Machine Learning: Gradient descent approximates function behavior using first-order Taylor expansion.

5. Worked Example: Approximating a Function

Approximate \(f(x,y) = \ln(1+x+y)\) near (0,0) using second-order Taylor series

  • \(f(0,0) = \ln(1) = 0\)
  • \(f_x = \frac{1}{1+x+y} \Rightarrow f_x(0,0) = 1\)
  • \(f_y = \frac{1}{1+x+y} \Rightarrow f_y(0,0) = 1\)
  • \(f_{xx} = -\frac{1}{(1+x+y)^2} \Rightarrow f_{xx}(0,0) = -1\)
  • \(f_{yy} = -\frac{1}{(1+x+y)^2} \Rightarrow f_{yy}(0,0) = -1\)
  • \(f_{xy} = -\frac{1}{(1+x+y)^2} \Rightarrow f_{xy}(0,0) = -1\)
  • Second-order approximation: \[ f(x,y) \approx 0 + (x+y) + \frac{1}{2}[-x^2 - 2xy - y^2] = x + y - \frac{1}{2}(x+y)^2 \]

6. Summary

  • Taylor’s theorem generalizes linear approximation to higher-order derivatives.
  • First-order = linearization, second-order = quadratic approximation.
  • Used in optimization, numerical methods, physics, engineering, and ML.
  • Important for understanding local behavior of multivariable functions.
